# WebApp - Patient Visit Tracker Frontend

React frontend for WoundTech patient visit tracking system.

## Tech Stack

- React 18 with TypeScript
- Vite
- Axios
- CSS3

## Features

- **Authentication**: Secure login required
- View clinicians and patients lists
- Create new visits
- Display visits table
- Filter visits by clinician or patient
- Auto-logout on token expiration

## Default Login

```
Email: super@woundtech.com
Password: SuperPassword123
```

See [AUTHENTICATION.md](AUTHENTICATION.md) for detailed authentication docs.

## Setup

npm install

## Development

npm run dev

App runs at http://localhost:5173
Backend API must be running on http://localhost:3000

## Build

npm run build
